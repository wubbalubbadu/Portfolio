üí∞ Market

MARKET is a website for Northwestern students to request and sell second-hand items. This full stack application is built with the MERN-Stack and uses Redux for state management, styled with Material-UI.

Features and Build Stack

Features responsive frontend design with CRUD functionalities, search/filtering/pagination, account page, user portal, uploading images, and adding to favorites using ReactJs and Redux, styled with Material-UI.

The frontend is consuming an ExpressJs Rest API with many End-Points which allows user to can register, login, create, update, and delete listings and requests.
MongoDB is used for database and deployed to AWS EC2.

Used Cloudinary to manage image uploads, incorporated JWT and Google authorizations using Passport.js.

SKUY is a community-based social media app and news aggregation platform. I am part of the developer team and I am responsible for the frontend and backend development.

Main Features

The main features of the app are communities, posts, and a scraping platform that aggregates news from various sources. The app is built with React Native and the backend is implemented with Flask and PostgreSQL, deployed using Heroku. News aggretation is accomplished using Firebase.
